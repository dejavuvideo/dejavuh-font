# DEJAVU:H Typeface

## Access

This repository does not provide downloadable font files.

DEJAVU:H is an experimental typeface combining Latin and Hangul structures.
It is currently presented as an artwork and research-based typographic system.

The system reconstructs Hangul using a Latin-based structural logic, exploring perception, reading, and linguistic transformation.

For usage inquiries, please contact the author.

---

## About

DEJAVU:H is an original typeface project developed by Kwangkee Lee.
It applies a systematic composition method combining Latin consonants and Hangul vowels.

This typeface is an independent work and is not related to the DejaVu font project.
